# 연산자

> 비트 연산자랑 시프트 연산은 프밍에서 했을테니까 패스

> 논리연산도 알지? (&& = and , || = or , ! = not)

> 증감 연산자 (a++ , ++a , a-- , --a) 알꺼고

- String 연산자

> Example Code

	var a = "123";
	var b = "45";

	var c = 123;
	var d = 45

	a + b = "12345"
	a + c = "123123"
	c + d = 168

> 그니까 정수덧셈 빼고는 다 문자열처럼 더해짐 ㅇㅋ?

- 비교연산

> (> , < , >= , <= , != , ==) 이거는 프밍에서 했으니까 다알거고

> 다 똑같은데 다른게 하나있음

> Example Code

	var a = "1"
	var b = 1;

	console.log(a == b)

> true

	console.log(a === b)

> false

> JS는 변수가 var밖에 없어서 형(type)까지 체크할려면 === 써야됨 (== 쓰면 int랑 string이랑 같다고 함 개꿀 ㅎㅎ)
